# Native-Ranked


This is the successor of the old and dead Native MCTiers app, as I lost hope with mctiers

Its still in Alpha so its not really finished yet



# Libraries being used

[Naett](https://github.com/erkkah/naett) (for HTTP)

[Raylib](https://www.raylib.com/) (for Windowing and Drawing)

[RayGUI](https://github.com/raysan5/raygui) (for making a decent GUI in Raylib)

[yyjson](https://github.com/ibireme/yyjson) (for JSON parsing)

[zthread](https://github.com/z-libs/zthread.h) (for cross-platform multithreading)
